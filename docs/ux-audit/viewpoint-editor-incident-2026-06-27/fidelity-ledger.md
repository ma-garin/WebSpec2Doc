# 観点編集モーダル 視覚忠実度台帳

## 視覚基準

- 正式な基準画像: `02-selected-modal-reference.png`（1366×768）
- 対象仕様: 案2「セクションナビ型大型モーダル」
- 補助資料: `../../assets/viewpoint-management-concept.png` は色・書体・一覧密度の参考に限る。常設編集パネルのレイアウトは採用しない。
- 実装トークン: 既存の `tokens.css` と `app.css` を再利用する。

## 比較結果

| 確認点 | 基準 | 実画面の証跡 | 判定 |
|---|---|---|---|
| モーダル外形 | 最大幅1120px、上下24px、左右32px以上 | 1366×768で1120×720、1024×768で左右32px | PASS |
| 内部構成 | 左セクションナビ、中央フォーム、右検証レール | 3領域を維持し、中央は1024pxでも480pxを確保 | PASS |
| スクロールと操作 | 中央だけをスクロールし、ヘッダーとフッターを固定 | 保存・破棄・削除を常時表示。ページ全体の横溢れなし | PASS |
| 状態表示 | 完了・未完了・エラーを文言とアイコンで表示 | 左ナビ、ヘッダー、右レールに各状態を表示 | PASS |
| 配色・文字・境界 | 既存WebSpec2Docの青、白、グレー、Noto Sans JP | 既存トークンを使用し、新規の架空ブランド色なし | PASS |
| 実データ | 版、継承、検証、保存、適用情報のみ | 既存APIの版・継承・割当数を表示。架空のサイト数等なし | PASS |
| 1024px | 左右レールを縮小し、中央フォームを優先 | 144px / 480px以上 / 172px、横スクロールなし | PASS |
| 主要コピー | 4セクション、破棄、削除、保存、次版作成 | 指定コピーの欠落・名称変更なし | PASS |
