## 0. 共通実装規約（全レーン必読・違反はレビューで差し戻し）

1. **XSS**: 外部由来文字列（テストtitle・error・URL・ユーザー入力）をHTMLに埋める時は必ず `escHtml()`（フロント）／`html.escape()`（Python）を通す。`innerHTML` への未エスケープ連結は禁止。
2. **入力検証**: 新規/変更APIのパラメータはホワイトリスト検証（例: `device in ("pc","mobile")`）。domain系は既存 `_valid_domain()` パターンを踏襲。検証失敗は 400 + 日本語エラー。
3. **不変性**: Python は frozen dataclass / タプル。既存 dict を書き換えず新 dict を返す。
4. **エラー処理**: 握りつぶし禁止。ユーザー向けは日本語メッセージ、サーバ側は `logger.exception`。
5. **evidence-only**: 実測にない値・状態を表示しない。不明は「不明/未確認」と明示。
6. **CSP**: `web/security.py:14-23` の `script-src 'self' 'unsafe-inline'` を**変更しない**（CDNをallowlistに足さない）。外部JSは `static/vendor/` に同梱して 'self' で読む。
7. **コミット**: 型プレフィックス（feat/fix/test/docs/perf/chore）、attribution行なし。UI変更は `make verify-ui` 後にコミット。
