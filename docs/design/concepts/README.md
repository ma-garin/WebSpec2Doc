# デザイン刷新 — 2026年版コンセプト候補（5案）

作成日: 2026-07-15 / ステータス: 方向性選定待ち

## 目的

現行UI（10年前風のシンプルなMaterial系デザイン）を、コンセプト・画面構成・機能を変えずに
2026年水準のビジュアルへ更新するための方向性候補。日本の商用ツール的な派手さや
装飾アニメーションは全案とも排除している。

`docs/design/ui-redesign-plan.md`（画面デザイン刷新計画）の前段にあたる資料であり、
ここで方向性を1案選定したのち、`static/tokens.css` を単一情報源としてトークン更新 →
`app.css` / `app-components.css` へ展開する流れを想定する。

## モックの前提

- 対象画面: ダッシュボード（ホーム）。`templates/partials/view-dashboard.html` の
  DOM構成（ヒーロー / モードカード / 解析履歴サマリ+テーブル）を全案で共通に再現
- 各HTMLは自己完結（CSS内蔵・外部依存なし）。1440×900・2xでスクリーンショット化
- フォントはNoto Sans JP系（案5の見出しのみNoto Serif JP）

## 5案の比較

| # | 案 | ファイル | 要点 | 移行コスト |
|---|-----|---------|------|-----------|
| 1 | **Tonal Calm**（MD3正統進化） | `concept-1-tonal-calm.*` | 現行ブルーDNA維持。トーナルサーフェス、ピル形状、ボーダーレス+極薄シャドウ、radius 12〜24px | 最小（トークン更新が中心） |
| 2 | **Neutral Minimal**（開発者ツール系） | `concept-2-neutral-minimal.*` | ほぼ無彩色+青1アクセント。シャドウ廃止・1pxボーダー、コンパクト密度 | 小〜中 |
| 3 | **Soft Depth**（レイヤード） | `concept-3-soft-depth.*` | 淡ブルーグレー背景に白カードを多層ソフトシャドウで浮かせる。半透明サイドバー | 中 |
| 4 | **Enterprise Grid**（Carbon系） | `concept-4-enterprise-grid.*` | 直角・ダークサイドレール・左アクセントバー・高密度グリッド。業務QAツールの堅牢さ | 中〜大（形状・密度が変わる） |
| 5 | **Warm Editorial**（ドキュメントツール系） | `concept-5-warm-editorial.*` | 温白×インク、セリフ見出し、広い余白。「文書生成ツール」らしい静けさ | 中（フォント追加あり） |

## 次のステップ

1. 5案から方向性を選定（組み合わせ・調整の指定も可）
2. 選定案で `static/tokens.css` のトークンを更新し、ダークテーマ対応を定義
3. `ui-redesign-plan.md` のフェーズ計画に沿って実装 → `make verify-ui` で検証
